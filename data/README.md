# data — Jeu de données source

`heart_disease_health_indicators_BRFSS2015.csv` — 253 680 lignes × 22 colonnes.

- **Source** : [Kaggle — Heart Disease Health Indicators Dataset](https://www.kaggle.com/datasets/alexteboul/heart-disease-health-indicators-dataset)
- **Origine** : CDC, *Behavioral Risk Factor Surveillance System* (BRFSS), édition 2015
- **Cible** : `HeartDiseaseorAttack` (0/1) — 9,42 % de positifs
- **Qualité** : aucune valeur manquante ; 23 899 doublons exacts, **conservés
  volontairement** (voir la note dans `00_documentation/2_description_donnees.md`)

Ces données sont publiques et anonymes : elles sont versionnées dans le dépôt
pour garantir la reproductibilité du projet.
